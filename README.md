Antes de qualquer coisa, rode:

```bash
$dart pub get
```


Para rodar o programa, vá para a raíz do projeto e coloque:

```bash
$dart run
```

Para rodar os testes, vá para a raíz do projeto e coloque:

```bash
$dart test
```

Padrões de projetos utilizados:

- singleton (Baralho)
- strategy (PegarCartaStrategy)
